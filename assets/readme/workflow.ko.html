🔗 자동 연결 파이프라인
대표 개발 흐름 — 5단계, 자동으로 이어집니다
각 커맨드가 다음 단계로 자동 넘어갑니다. 수동으로 연결할 것이 없어요.
1
📋
/plan
계획 세우기
AI가 요구사항을 정리해 plan.md를 만듭니다. 사람이 승인한 뒤에야 코드 작성이 시작돼요.
승인 게이트
자동
2
🧪
/tdd
테스트 먼저
실패 테스트 먼저 작성 → 통과하도록 구현 → 리팩터. TDD 에이전트가 RED→GREEN 사이클을 스스로 돌려요.
RED → GREEN
자동
3
🔍
/code-review
코드 검토
Codex·Gemini·보안 전문가·아키텍트 4명이 병렬로 리뷰. 혼자 보면 놓칠 문제를 잡아냅니다.
리뷰어 4명 병렬
자동
4
✅
/handoff-verify
한 번에 검증
계획 대비 구현 일치 확인, 테스트 실행, 보안 훅 점검까지 한 번에. 빠진 것 없이 올그린일 때만 통과.
올그린 게이트
자동
5
🚢
/commit-push-pr
커밋·PR 올리기
커밋하고, 푸시하고, 계획과 리뷰 내용이 담긴 PR 설명을 자동 작성해서 올립니다.
